BiblioteCAT

This is a project that consumes the OpenLibrary API.
The idea is to practice proper API integration and build an interactive website. BiblioteCAT allows you to search for books by different types (title, author, subject), and view details like title, publication year, description, and author information. It also features a wishlist — you can add and remove books in real time. The data is saved in your browser's local storage, so no login is required. And last but not least, the “CAT” in the name comes from the cat button on the homepage, just a small touch of fun for this project.

Cat Button

The cat button, as you might expect, is a button — but with a twist. When you press it, your search will be taken over by a cat. The results? Probably something a cat would search for — not you. But hey, it’s a cat button, so…

Search Input

The search input is where you type what you're looking for — a title, author, or subject. Just select the type of search from the dropdown. I recommend searching in English. Since I don’t own the API, I can’t guarantee great results in other languages.
